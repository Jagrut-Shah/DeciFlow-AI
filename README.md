# DeciFlow AI

DeciFlow AI is a modern intelligent application.

## Project Structure

* **`backend/`**: Backend services and APIs.
* **`frontend/`**: Frontend web application.
* **`ml/`**: Machine learning models, pipelines, and training scripts.
* **`infra/`**: Infrastructure as code (IaC) and deployment configurations.
* **`docs/`**: Project documentation, architecture diagrams, and design records.
* **`scripts/`**: Utility and operational scripts (e.g., migrations, seeding).
* **`tests/`**: End-to-end tests and integration testing frameworks.

## Getting Started

1. Copy `.env.example` to `.env` and fill in necessary environment variables.
2. Ensure you have Docker and Docker Compose installed.
3. Start the project using Docker Compose:
   ```bash
   docker-compose up -d
   ```
