# DeciFlow AI: API Contract

This document outlines the core API integrations between the Next.js frontend and the FastAPI backend.

## Endpoints

### POST /upload
Uploads a new dataset for processing and analysis.

* **Request Format**: 
  * Header: `Content-Type: multipart/form-data`
  * Body: `file` (File object)
* **Response Format**:
  * `200 OK`
  * Returns the metadata and status of the uploaded file.
* **Example Response**:
  ```json
  {
    "status": "success",
    "file_id": "data_12345",
    "filename": "q1_sales.csv",
    "message": "File successfully uploaded and queued for processing."
  }
  ```

### GET /insights
Retrieves analyzed insights from processed data.

* **Request Format**: 
  * Query Parameters: `dataset_id` (string, required)
* **Response Format**:
  * `200 OK`
  * Returns an array of insights generated by the Insight Agent.
* **Example Response**:
  ```json
  {
    "dataset_id": "data_12345",
    "insights": [
      {
        "category": "performance",
        "description": "Sales increased by 15% in the Northeast region."
      }
    ]
  }
  ```

### GET /predictions
Fetches machine learning forecasts and predictive models based on current trends.

* **Request Format**: 
  * Query Parameters: `dataset_id` (string, required), `timeframe` (string, optional)
* **Response Format**:
  * `200 OK`
  * Returns numerical forecasts and confidence intervals.
* **Example Response**:
  ```json
  {
    "metric": "future_sales",
    "forecast": [1200, 1350, 1400],
    "confidence": 0.89
  }
  ```

### GET /decisions
Retrieves recommended actions from the Decision Engine.

* **Request Format**: 
  * Query Parameters: `dataset_id` (string, required)
* **Response Format**:
  * `200 OK`
  * Returns a ranked list of recommended decisions alongside their explanation.
* **Example Response**:
  ```json
  {
    "decisions": [
      {
        "rank": 1,
        "action": "Increase marketing spend in Northeast",
        "score": 92.5,
        "reasoning": "High predicted ROI based on recent Q1 trends."
      }
    ]
  }
  ```

### POST /simulation
Runs a scenario simulation based on proposed variables.

* **Request Format**: 
  * Header: `Content-Type: application/json`
  * Body: `{"variables": {"marketing_budget": 50000}}`
* **Response Format**:
  * `200 OK`
  * Returns simulated outcomes and impact metrics.
* **Example Response**:
  ```json
  {
    "scenario_id": "sim_999",
    "predicted_impact": {
      "revenue": "+12%",
      "cost_acquisition": "-5%"
    }
  }
  ```

### POST /chat
Interacts with the AI Copilot for conversational data queries.

* **Request Format**: 
  * Header: `Content-Type: application/json`
  * Body: `{"message": "What factors caused the drop in Q3?"}`
* **Response Format**:
  * `200 OK`
  * Returns a natural language response generated by the AI reasoning layer.
* **Example Response**:
  ```json
  {
    "reply": "The drop in Q3 was primarily driven by supply chain delays detected in August, lowering fulfillment rates by 8%."
  }
  ```
